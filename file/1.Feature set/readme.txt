"data set.xlsx" contains the aggregate data characteristics of traffic for a total of three categories, 30 for each category, for a total of 90 data.

'"FTP", "VoIP" and "Stream" are three files, each containing the overall PCAPNG file of the three traffic types, and are divided into 30 subfiles.

There are five Python programs for different functions, three for reads and two for calculations.As long as you change the path code, you can complete the operation of different machines.
